# Week 6 Example 2 — Free Roam Top-Down with Boss Battle: Papers Please

## Setup and Interaction Instructions

To run the sketch locally, open `index.html` in Google Chrome using Live Server.

**Controls:**

- Move: WASD
- Shoot: Spacebar (shoots in the direction you last moved)
- B: Skip to boss fight (testing only)
- Restart: R (after win or game over)

Explore the Papers Please world and survive enemy waves as you move north, then enter the boss zone to fight the giant guard. Watch the minimap to track enemies off screen.
